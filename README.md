# Harbors Official

Public repository for Harbors LLC — company site and public-facing assets for the Harbors Urban OS project.

This repo contains the static company landing page and demo UI (`index.html`) used for public relations and press. Internal code, APIs, and sensitive assets are stored in private repositories.

## What this repo contains
- `index.html` — public landing page and map demo for coin-locker / urban infrastructure information.
- `README.md`, `LICENSE` — repository metadata.

## Publishing (GitHub Pages)
To publish this repository as a project site:

1. Push the repository to `https://github.com/harbors-jp/harbors-official`.
2. In the repository Settings → Pages, set the Source to `main` branch and `/ (root)`.
3. The site will be available at `https://harbors-jp.github.io/harbors-official/` after a few minutes.

## Notes
- This repository is public-facing. Do not commit secrets, API keys, or private data here.
- For development, use private repositories: `harbors-core`, `harbors-api`, `harbors-data`, etc.
